状语从句
(Adverbial Clause)
keyboard_arrow_down

从句类型引导词说明例子
时间状语从句 (Time Adverbial Clause)when, while, as, since, before, after, until, as soon as, once, whenever...描述事情发生的时间我们一到那儿，就开始下雨了。（as soon as 引导的时间状语从句）； 他在等地铁的时候，突然接到了一个电话。（when 引导的时间状语从句）； 不管你什么时候来，我都会在这儿等你。（无论疑问词引导的时间状语从句）
地点状语从句 (Place Adverbial Clause)where, wherever描述事情发生的地点无论你去哪里，我都会跟着你。（wherever 引导的地点状语从句）； 我们在街角碰头。（where 引导的地点状语从句）
原因状语从句 (Reason Adverbial Clause)because, since, as, for, seeing that, considering that, now that描述事情发生的原因由于天气太热，他决定呆在家里。（because 引导的原因状语从句）； 既然你有时间，为什么不去看看电影呢？（since 引导的原因状语从句）
结果状语从句 (Result Adverbial Clause)that, so that描述事情的结果他太累了，以至于走不动路了。（so that 引导的结果状语从句）； 他学习非常努力，以至于最后获得了学院的最高奖学金。（that 引导的结果状语从句）
条件状语从句 (Condition Adverbial Clause)if, even if, only if, provided, providing, as long as, unless, on condition that描述一个条件成立的情况如果你想成功，你必须要努力工作。（if 引导的条件状语从句）； 除非天气很好，否则他不会去爬山。（unless 引导的条件状语从句）
方式状语从句 (Manner Adverbial Clause)as, like, just as, in the way, in the same way描述事情发生的方式或方式的相似性你必须要做到和我一样努力才能成功。（like 引导的方式状语从句）； 正如他所说的那样，努力就是胜利。（as 引导的方式状语从句）
比较状语从句 (Comparison Adverbial Clause)than, as...as, the same...as描述比较两件事情、人或物的差异或相似之处他的英语比我好多了。（than 引导的比较状语从句）； 她和我一样年轻。（as...as 引导的比较状语从句）
目的状语从句 (Purpose Adverbial Clause)in order that, so that, lest描述事情发生的目的我留了一些钱给你，以便你在需要的时候可以拿来用。（in order that 引导的目的状语从句）； 我把窗户打开，以便新鲜空气可以进来。（so that 引导的目的状语从句）； 我把门反锁了，以免有人进来打扰我。（lest 引导的目的状语从句）
