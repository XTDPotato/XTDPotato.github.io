公式 导数 不定积分 定积分
f(x) = k (常数) f'(x) = 0 F(x) = kx + C ∫a^b k dx = k(b - a)
f(x) = x^n (n为常数) f'(x) = nx^(n-1) F(x) = (x^(n+1))/(n+1) + C ∫a^b x^n dx = [(b^(n+1))/(n+1)] - [(a^(n+1))/(n+1)]
f(x) = e^x f'(x) = e^x F(x) = e^x + C ∫a^b e^x dx = e^b - e^a
f(x) = ln(x) f'(x) = 1/x F(x) = x ln(x) - x + C ∫a^b ln(x) dx = (b ln(b) - b) - (a ln(a) - a)
f(x) = sin(x) f'(x) = cos(x) F(x) = -cos(x) + C ∫a^b sin(x) dx = -cos(b) + cos(a)
f(x) = cos(x) f'(x) = -sin(x) F(x) = sin(x) + C ∫a^b cos(x) dx = sin(a) - sin(b)
f(x) = tan(x) f'(x) = sec^2(x) F(x) = ln|sec(x)| + C ∫a^b tan(x) dx = ln|sec(a)| - ln|sec(b)|
f(x) = cot(x) f'(x) = -csc^2(x) F(x) = ln|sin(x)| + C ∫a^b cot(x) dx = ln|sin(b)| - ln|sin(a)|

求导公式：

常数求导：(c)' = 0，其中c是常数。

幂函数求导：(x^n)' = nx^(n-1)，其中n是实数，x是自变量。

指数函数求导：(e^x)' = e^x，其中e是自然对数的底数。

对数函数求导：(ln(x))' = 1/x，其中ln表示以e为底的对数。

三角函数求导：(sin(x))' = cos(x)，(cos(x))' = -sin(x)，(tan(x))' = sec^2(x)，其中sin、cos和tan表示正弦、余弦和正切函数。

反三角函数求导：(arcsin(x))' = 1/√(1-x^2)，(arccos(x))' = -1/√(1-x^2)，(arctan(x))' = 1/(1+x^2)，其中arcsin、arccos和arctan表示反正弦、反余弦和反正切函数。

链式法则：若y = f(g(x))，则y' = f'(g(x)) * g'(x)。

积分公式：

幂函数积分：∫x^n dx = (x^(n+1))/(n+1) + C，其中n不等于-1，C为常数。

指数函数积分：∫e^x dx = e^x + C。

对数函数积分：∫(1/x) dx = ln|x| + C。

三角函数积分：∫sin(x) dx = -cos(x) + C，∫cos(x) dx = sin(x) + C，∫sec^2(x) dx = tan(x) + C。

反三角函数积分：∫(1/√(1-x^2)) dx = arcsin(x) + C，∫(-1/√(1-x^2)) dx = arccos(x) + C，∫(1/(1+x^2)) dx = arctan(x) + C。

分部积分法：∫u dv = uv - ∫v du，其中u和v都是关于自变量的函数。
